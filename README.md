# Project Title

DR twitter, as side project trying to replicate building a twitter like app using django rest framework
and react for the frontend

## Getting Started

clone the repo

### Prerequisites

The frontend runs on react so you will need npm to install all required frontend librariers.
To install backend librarier you will need to have pip install 

to get node
https://www.npmjs.com/get-npm



### Installing

create a virtualenv and run the command below

```
pip install -r requirements.txt
```

To install the frontend librariers. Navigate to the frontend folder and run the command below

```
npm install
```
